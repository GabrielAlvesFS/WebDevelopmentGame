Sobre o Jogo:

Este é um Quiz que foi feito com o intuito de testar o seu conhecimento na área de Desenvolvimento Web.

Neste Quiz temos três personagens do Desenvolvimento Web que são: HTML, CSS e o Javascript. Em cada personagem será abordado um tema específico e nele você terá a missão de acertar as perguntas propostas para você!!!
